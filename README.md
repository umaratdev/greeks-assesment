# Project Name: Greeka Assignment

## How to Run the Development Server

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## How to Commit Changes

1. Use Commitizen for consistent commit messages:
   ```bash
   npm run commit
   ```

2. Follow the guided prompts to create a descriptive commit message.
